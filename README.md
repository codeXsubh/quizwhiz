# QuizWhiz

A dynamic Java Swing Quiz Application that fetches real-time MCQs from a public API.

## Features

- API-driven question loading (Open Trivia DB)
- Dynamic UI rendering with adaptive layout
- Timer per question
- 50-50 lifeline feature
- Automatic score calculation
- Maven-based dependency management
- Clean MVC-style structure

## Tech Stack

- Java 21
- Swing
- Maven
- Gson
- Java HTTP Client API

## Project Structure

src/main/java  
src/main/resources  
pom.xml  

## How to Run

1. Clone repository
2. Run `mvn clean install`
3. Run `Questions.java`
